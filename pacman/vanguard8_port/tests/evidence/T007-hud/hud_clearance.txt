T007 HUD overlay clearance note

Coordinate system:
- VDP-A Graphic 3 cells are 8x8 pixels, 32 columns by 24 rows.
- The left gutter is column 0.
- The clearance check uses the accepted T006 maze frame and the T007 static
  HUD cells listed in font_manifest.txt.

Occupied HUD cells:
- HIGH SCORE label: row 0, columns 11-14 and 16-20.
- High-score placeholder digits: row 1, columns 13-17.
- SCORE label: row 22, columns 13-17.
- Score placeholder digits: row 23, columns 13-17.
- Lives: rows 3-4, column 0.
- Fruit placeholder: row 6, column 0.

Clearance result:
- The centered text cells were chosen from the T006 cell scan runs that
  contain no dot or power-pellet pixels in the occupied cells.
- Column 0 cells are outside the 240 px centered maze render and remain in the
  left gutter.
- The HUD uses VDP-A color index 0 as transparent background with TP set, so
  all unused HUD pixels reveal VDP-B.

Known constraint:
- Vanguard 8 Graphic 3 currently fetches tile rows only for scanlines 0-191.
  The bottom score strip therefore uses rows 22-23 rather than the final
  physical 20 scanlines.
