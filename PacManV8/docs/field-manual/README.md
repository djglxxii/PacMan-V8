# Developer's Field Manual

Hard-won learnings from Vanguard 8 game ROM development. Each file in this
directory captures a non-obvious technique, gotcha, or pattern discovered
during development.

These entries are intended to be:

- **Searchable** — descriptive filenames, clear titles.
- **Self-contained** — each entry stands alone with context, insight, and
  example.
- **Reusable** — written broadly enough to apply to future V8 projects,
  ROM ports, and similar retro-console development work.

See `CLAUDE.md` rule 9 for the recording protocol.
